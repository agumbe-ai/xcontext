# Security policy

Please report vulnerabilities privately through GitHub Security Advisories for `agumbe-ai/xcontext`. Do not open a public issue containing exploit details, credentials, customer data, or unredacted artifacts.

The `development` branch and unreleased builds are unsupported. Security fixes target the latest released minor version.

Raw context may contain credentials and proprietary code. The safe default is redacted storage. Original storage must be explicitly enabled and requires encryption, access auditing, retention enforcement, and a production-grade storage adapter.
